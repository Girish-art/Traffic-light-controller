# Traffic Light Controller Using Arduino

## Project Overview
This project demonstrates a Traffic Light Controller using Arduino Uno. Three LEDs (Red, Yellow, and Green) are used to simulate a real traffic signal. The LEDs switch in sequence with predefined timing intervals to control traffic flow.
## Components Used
- Arduino Uno
- Red LED
- Yellow LED
- Green LED
- 220 Ohm Resistors
- Breadboard
- Jumper Wires
- USB Cable
